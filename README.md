# WebProgramlamaProje
Fitness Center Management 


## Proje Açıklaması
Bu proje, Web Programlama dersi kapsamında ASP.NET Core MVC kullanılarak geliştirilmiş bir spor salonu yönetim ve randevu sistemi projesidir.

## Projenin İçeriği
- Spor salonu yönetimi
- Antrenör yönetimi
- Üye ve randevu işlemleri sistemi
- Yapay zeka entegrasyonu 
- REST API kullanımı ile veritabanı iletişimi

## Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core,LINQ
- SQL Server
- HTML5,CSS3,JavaScript,jQuery
- Bootstrap 5
- C#
